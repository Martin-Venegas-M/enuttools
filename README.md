# enuttools

<!-- badges: start -->

<!-- badges: end -->

The goal of enuttools is to provides functions for processing and
analyzing time use surveys based on a stylized questionnaire. The
package focuses on the National Time Use Survey conducted by the
National Institute of Statistics of Chile, but it can also be applied to
related statistical products following a similar design.
